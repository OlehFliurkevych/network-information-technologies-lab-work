# Laboratory work for Network information technologies subject
 
This example app shows how to create a Spring Boot API and CRUD (create, read, update, and delete) its data with a React app.

**Technologies:**

* Java 21
* Spring Boot
* ReactJS
* MySQL database
* Okta
* Maven
